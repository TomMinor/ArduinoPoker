\subsection{Detailed Description}
The main class used for displaying things on the screen. 

The \hyperlink{namespaceGUI}{G\-U\-I} class includes functions to be called for most of the events that will happen in the game such as dealing a card to a player, receiving a bet from a player or adding a card to the public set. A unique example is the \char`\"{}show\-Winner()\char`\"{} function which returns a std\-::vector of Hands which must be burned outside of the \hyperlink{namespaceGUI}{G\-U\-I} class before a new round is started. There are also functions to return dynamically allocated instances of visual elements (e.\-g. cards, text) that will be drawn, updated, and destroyed by the \hyperlink{namespaceGUI}{G\-U\-I} class but can be controlled externally. Finally, there are the \hyperlink{classGUI_1_1DealerGUI_a479eecd663924d0616e516af9ca2679e}{update()} and \hyperlink{classGUI_1_1DealerGUI_a034f53fffe418f01eadd8e7aa7a74589}{draw()} functions which should be called about 30 times a second during gameplay and the \hyperlink{classGUI_1_1DealerGUI_aa4cca14de7ab4526b6fafc6c5f226397}{reset()} function which should be used when a new round is started.
