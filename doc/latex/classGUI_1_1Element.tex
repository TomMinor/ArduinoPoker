\subsection{Detailed Description}
The base class for any visual elements on the screen. 

It and any derived classes include functions to move the element to a point on the screen (optionally aligned with an edge of the screen) instantly or with cosine interpolation. The lifetime of the element (after which it will be destroyed automatically) can be set in the constructor.
