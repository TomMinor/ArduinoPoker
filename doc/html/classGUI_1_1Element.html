Detailed Description

The base class for any visual elements on the screen.

It and any derived classes include functions to move the element to a point on the screen (optionally aligned with an edge of the screen) instantly or with cosine interpolation. The lifetime of the element (after which it will be destroyed automatically) can be set in the constructor.

Parameters
_renA pointer to the SDL rendering context.
_texA pointer to the SDL texture to use.
_srcRectAn SDL rectangle that covers the part of the texture to use.
_destRectAn SDL rectangle that specifies the part of the screen to draw the previous rectangle to.
_orientationAn enum specifying which edge of the screen this element should face.
_originAn SDL point specifying the exact location this card should be created. Making sure this is in the centre of _destRect is handled by the ElementMaker class.
_lifetimeHow many update() calls this element should last until it is destroyed. Passing 0 makes this lifetime indefinite.

A drawable object.
